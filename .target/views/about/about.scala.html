About Computex RDF Validation

This project validates RDF data profiles in general.

RDF profiles can be RDF Data Cube datasets or any other profile made from a base ontology, a list of SPARQL query validators and a list of SPARQL update extension queries.

We have written a position paper which contains more information about the Computex validation process

The source of the project is at GitHub

It also validates RDF Data Cube data by executing the Integrity Constraints

This project has been made by WESO (WEb Semantics Oviedo)
